## Rencana Update: Efek Carousel Geser Panjang

Saya akan mengubah animasi dari "geser halus" menjadi **"Carousel Slide"** yang bergeser penuh (100% lebar), serta memperbaiki bug teknisnya.

### 1. Update CSS (`optimized_styles.css`)
Saya akan mengubah parameter animasi agar slide bergerak dari ujung ke ujung (seperti carousel pada umumnya), dan menghilangkan efek pudar (fade) agar terlihat solid saat bergeser.

*   **Jarak Geser**: Mengubah `translate3d(24px, ...)` menjadi `translate3d(100%, ...)` (dan `-100%`). Ini membuat slide masuk dari luar layar sepenuhnya.
*   **Opasitas**: Mengubah `opacity: 0` menjadi `opacity: 1` pada saat transisi (`is-entering` dan `is-exiting`). Ini memastikan slide tetap terlihat jelas selama proses pergeseran, tidak menghilang/transparan.

### 2. Update JavaScript (`optimized_script.js`)
Menerapkan perbaikan **Double RequestAnimationFrame** untuk memastikan browser merender frame awal animasi dengan benar, sehingga efek geser panjang ini dijamin berjalan mulus (tidak langsung "lompat" ke tengah).
